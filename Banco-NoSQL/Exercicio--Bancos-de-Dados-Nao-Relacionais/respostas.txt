1. Definição e Características

a) O que caracteriza um banco de dados não-relacional?

  um banco de dados nao recionais sao banco de dados NoSql ou seja sem sequencia eles sao mais flexiveis e podem ser usados com uma grande variedade de formatos

b) Quais são as vantagens desse tipo de banco em relação aos bancos
relacionais?

  os bancos relacionas ou Sql como o postgresql sao bancos de dados com uma sequencia e que sao melhores com uma maior quantidade de dados

c) Cite ao menos 3 situações práticas em que um banco de dados não-
relacional é mais indicado.

  em sites onde precisa de varios logins ou que tem muitas pessoas acessando simultaneamente

2. Tipos de Bancos de Dados Não-Relacionais

a) Explique as diferenças entre os quatro principais tipos:

  Orientados a Documentos
    estrutura flexivel
  Chave-Valor
    dados armazenados como pares simples: chave -> valor
  Colunas Amplas / Wide Column
    armazenam dados em colunas organizadas por familias de colunas
  Grafos
    conexao entre dados

b) Dê exemplos de ferramentas populares para cada tipo.

  Documentos = MongoDB
  Chave-Valor = Redis
  Colunas Amplas = Apache Cassandra 
  Grafos = Neo4j
